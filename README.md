Staff Leave Management System

A professional Leave Management System developed using Django. This project helps organizations manage staff leave requests efficiently through separate Admin and Staff dashboards.

⸻

🚀 Features

👨‍💼 Admin Module

* Admin Dashboard
* Manage Staff
* Add Staff
* Leave Management
* Approve / Reject Leave Requests
* Rejection Message System
* Manage Leave Types
* Profile Management
* Change Password
* Professional Responsive UI

⸻

👨‍💻 Staff Module

* Staff Dashboard
* Apply Leave
* Leave History
* Leave Tracking System
* Cancel Leave Request (Before Approval/Rejection)
* Leave Status Tracking
* Rejection Message View
* Profile Management
* Responsive Professional UI

⸻

⚙️ Installation

1️⃣ Clone Repository

git clone https://github.com/your-username/Staff_leave_management_system.git

⸻

2️⃣ Navigate to Project

cd Staff-Leave-MS-Django-Python

⸻

3️⃣ Create Virtual Environment

python -m venv venv

⸻

4️⃣ Activate Virtual Environment

Mac/Linux

source venv/bin/activate

Windows

venv\Scripts\activate

⸻

5️⃣ Install Dependencies

pip install -r requirements.txt

⸻

6️⃣ Run Migrations

python manage.py migrate

⸻

7️⃣ Create Superuser

python manage.py createsuperuser

⸻

8️⃣ Run Server

python manage.py runserver
